# To-do Card

A simple to-do card web app built with HTML, CSS, and JavaScript.

## Project Overview

This project provides a lightweight task card interface that allows users to manage to-do items in their browser. It includes a clean design and basic interactive functionality for adding, editing, and removing tasks.

## Features

- Add new to-do cards
- Remove existing tasks
- Responsive layout for desktop and mobile screens
- Built with vanilla HTML, CSS, and JavaScript

## Files

- index.html - Main HTML page for the application.
- styles.css - Styling for layout, cards, and responsive design.
- script.js - JavaScript logic for creating and managing to-do cards.
- READMe.md - Project documentation.

## Getting Started

1. Clone or download the repository.
2. Open index.html in your web browser.

## Usage

- Type a new task into the input field.
- Click the add button to create a new card.
- Use the delete button on a card to remove it.

## Notes

- No build tools or dependencies are required.
- This is a beginner-friendly project ideal for learning basic web development.
